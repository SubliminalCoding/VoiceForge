Setup in 5 steps

  1. Download and install

    Open the DMG file above, then drag VoiceForge into your Applications folder.

  2. Bypass the macOS warning

    On first launch, macOS will warn about an unidentified developer. Right-click the app, click Open, then click Open again to confirm.

  3. Get your free Groq API key

    Create an account at console.groq.com and generate an API key. It's free.

  4. Paste your API key

    Open VoiceForge settings and paste your Groq API key into the API key field.

  5. Set your hotkey and start dictating

    Pick a global hotkey in settings, then press it anywhere to start talking. Your words hit the clipboard in seconds.

Frequently asked questions
  • Why does macOS show a warning?
    VoiceForge is distributed outside the Mac App Store, so macOS Gatekeeper flags it as unidentified. This is normal for indie Mac apps. Right-click the app, choose Open, then confirm. You only need to do this once.
  • How do I get a Groq API key?
    Go to console.groq.com, sign up for a free account, and create an API key from your dashboard. Groq offers a generous free tier that covers normal usage.
  • Is my data private?
    Yes. VoiceForge processes audio locally on your machine using your own API key. Your recordings and transcriptions are never sent to VoiceForge servers. You control your data completely.
