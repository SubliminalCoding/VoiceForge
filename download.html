Setup in a few steps

  1. Download and install

    Open the file you downloaded and follow your platform's normal install flow (drag to Applications on Mac, run the installer on Windows, etc.).

  2. Bypass the first-launch macOS warning

    Apple-signed but not yet notarized, so Gatekeeper will warn on first launch. Right-click the app, click Open, then click Open again to confirm. You only need to do this once.

  3. If SmartScreen warns you, click "More info"

    VoiceForge is signed by an SSL.com certificate, but Windows builds reputation per-publisher over time. If SmartScreen blocks the installer, click More info and then Run anyway. This warning will go away automatically as more people install.

  4. Get your free Groq API key

    Create an account at console.groq.com and generate an API key. Free tier covers normal personal use.

  5. Paste your API key

    Open VoiceForge settings and paste your Groq API key into the API key field.

  6. Set your hotkey and start dictating

    Pick a global hotkey in settings, then press it anywhere to start talking. Your words hit the clipboard in seconds.

Frequently asked questions

Why does macOS show a warning on first launch?
VoiceForge is signed with an Apple Developer ID but not yet notarized through Apple's automated checks. Gatekeeper still flags signed-but-not-notarized apps as a precaution. Right-click the app, choose Open, then confirm. One-time only. Notarization is on the roadmap.
Why does Windows show a SmartScreen warning?
VoiceForge is signed by SSL.com, but Windows SmartScreen builds publisher reputation incrementally as more users install signed software. The warning disappears automatically once enough installs accumulate. Click More info → Run anyway in the meantime.
How do I get a Groq API key?
Go to console.groq.com, sign up for a free account, and create an API key from your dashboard. Groq's free tier covers normal personal usage.
Is my data private?
Yes. VoiceForge sends your audio directly from your machine to Groq using your own API key. Recordings and transcriptions are never sent to VoiceForge servers. You control your data.
